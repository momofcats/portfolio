News Explorer

Javascript Node.js Express.js React.js MongoDb

News explorer is a MERN stack application that fetches and displays articles from the News API.

After the users are authenticated, they are able to bookmark articles and view on their profile page.

This app was created with Create-React-App. The development process was done in 2-week agile sprints. The back-end is built in Node/Express.js and served on AWS(EC2) cloud server. User data, credentials, and saved articles are stored in MongoDb.

Around the U.S.

Javascript Node.js Express.js React.js MongoDb

Around the U.S. is an instagram-like content sharing app. This is a responsive single page React app with router navigation. Server side API is powered by Express.js server and hosted on Heroku.

The app features user profile registration and login. Authenticated users are redirected to the main page and are able to add pictures to the gallery and see the cards created by other users. Additionally the users can like/dislike or remove pictures.
